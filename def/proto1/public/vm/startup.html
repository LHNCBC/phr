The Virtual Machine (VM) PHR

Download the VM

The NLM PHR system is available for download in the form of a Virtual Machine (VM). If you would like to download this implementation, you will first need to install a VM manager on your system. We used Oracle's Virtual Box VM Manager to create the VM.

To download the VM, we require that you first read and agree to the License Agreement. Please acknowledge that you have read the agreement and agree to it by clicking here.
You may now download the VM file and import it into your manager. If you are using Oracle's Virtual Box, use the "Import Appliance" option from the File pulldown menu on the top menubar of the manager window. You may review the License Agreement again if you wish.

Once you've imported the VM, use the VM Manager to open it up. Several startup steps will be performed automatically and you will be presented with a Firefox browser window displaying the login page for the PHR system.

See the Notes page for a description of those steps as well as other startup info you will need.
